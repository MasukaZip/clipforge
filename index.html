ClipForge Publisher

O ClipForge Publisher ajuda criadores autorizados a preparar vídeos curtos, legendas, capas e descrições e a publicar o MP4 final no TikTok por meio da Content Posting API oficial.

Fluxo completo de ponta a ponta

1

Entrar com TikTok

O usuário abre o ClipForge local, clica em Conectar TikTok e autoriza o app pelo Login Kit usando PKCE.

2

Preparar o vídeo

O usuário seleciona um MP4 criado no ClipForge, revisa legenda, hashtags e configura a privacidade permitida pela conta.

3

Publicar com API oficial

O app envia o arquivo pela Content Posting API Direct Post e salva o publish_id retornado pelo TikTok.

4

Consultar status

O usuário acompanha PROCESSING, PUBLISH_COMPLETE ou FAILED pela API de status de publicação.

Escopos solicitados

Escopos mínimos para evitar incompatibilidade

O ClipForge Publisher solicita apenas os escopos necessários para login e postagem direta. Produtos e escopos não usados devem ser removidos do TikTok Developer Portal antes do reenvio.

user.info.basic Identifica a conta autorizada depois do login.
video.publish Publica o MP4 selecionado pelo usuário via Direct Post.

Callback OAuth

Como o login é concluído

A URL de redirecionamento configurada no TikTok Developer Portal deve ser esta página pública. Quando o TikTok retorna com code e state, a página encaminha os parâmetros para o backend local do ClipForge, onde o token é trocado com segurança.
